LinRPC

Spring Boot + gRPC microservice suite with real service discovery (Etcd/ZooKeeper), a REST gateway, and a demo script. It runs with full observability and ships Docker/Kubernetes deployments, so you can spin it up locally or in a cluster and see every call, metric, and strategy decision in action.

goTrojans

A matchmaking web app I built to help USC students quickly find partners for last-minute activities nearby. Users can post what they're looking to do, see others' requests on a map, hop into a group, chat, and wrap up with ratings. Under the hood it runs a TypeScript/NestJS backend with Postgres + Redis for real-time matching and a Next.js frontend that feels like a US consumer app.

Events around

You can search events, buy tickets, share to your friends and more. Call a list of APIs to get the data like Google map, ticketMaster, Spotify, etc. It's a small course project but it's really fun and can lead you to a lot of interesting ideas.
